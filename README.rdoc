== Questiona

Questiona is a light version of Quora, a clone.

I focused on back end core functionality more than front end 
and UI.


Features:

Users can follow other users.
Users can follow questions.
Users can answer questions.
Users can upvote answers.
Every activity gets log.
Each log becomes a notification if it fits the criteria.
Users have many notifications.

With de activity logs we create the user feed.
